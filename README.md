# Runsoc Geoguesser Clone

Javascript Geoguesser clone used for diversity fest 26T1. 

Unfortunately a lot of the features are hardcoded including
- Images, which you'll need to code in the coordinates and manually change the titles to numbers
- Score/difficulty threshold

Apologies for the lack of comments for the CSS, a lot of that was admittely cut and paste from online tutorials

The Javascript is powered by the Leaflet library originally developed by Vladimir Agafonkin: https://leafletjs.com/
